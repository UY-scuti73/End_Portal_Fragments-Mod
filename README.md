Ever think getting to the end was not much of a challenge? Well this mod forces players to explore all across your minecraft world to obtain unique eyes to unlock the portal. Most are found in various vanilla structures and obtained similarly to armor trims. This provides a unique and fun challenge for players!

# Portals
There are 8 new portals that take the place of some vanilla end portal frames in the stronghold. So every stronghold will contain 4 vanilla end portal frames, and 8 new and unique ones with their respective eyes. Only the eye of ender can direct one to a stronghold and fill the normal end portal frames like in vanilla.

# Eyes

There are 8 eyes that correspond to their portals found throughout the world. 
The loot table probability is lenient enough t one should spawn per structure. 
However, with bad luck it could be two or three. These parameters can be changed with a config file.

Ruined Eye - Elder Guardian Kill

Golden Eye - Piglin Bartering

Ancient Eye - Trail Ruins

Dark Eye - Ancient City

Withered Eye - Wither Kill

Gusty Eye - Trial Chamber Ominous Vault

Wealthy Eye - Woodland Mansion

Sandy Eye - Desert Temple

# Config

You can modify the config file for drop/loot chances. The values go from 0-1 
(0 being they don't spawn and 1 being the most). For most structures 1 means that they will always 
spawn in that chest or from that drop. However 1 just makes it very common for the trial chambers ominous vault, 
trail ruins, and piglin bartering. This is so you can still obtain other items from these loot tables.

For Fabric 26.1+ there is a second config file to modify the delay for replacing the end portals with 
the custom ones. If it is too small the server can try to replace them before the chunk loads fully, but too 
long and it might be noticable. The default is 40 ticks (2 seconds) and should be fine for all intensive purposes, 
however if a server is REALLY laggy it may need to be incremented.
